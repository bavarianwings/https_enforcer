Loading the Extension
---------------------

#. Login into the TYPO3 backend (administration) using the `http://www.mydomain.org/typo3/
   <http://www.mydomain.org/typo3/>`__  URL.
#. In the core module menu select ADMIN TOOLS > Extensions.
#. In the top-left Menu drop-down select **Get Extensions**.
#. Search for “https_enforcer”
#. In the search result click on the download button next to Page HTTP/HTTPS Enforcer.
#. Agree to make database updates.
#. In the top-left Menu drop-down select Installed Extensions.
#. Make sure one of the entries reads Page HTTP/HTTPS Enforcer.
