User manual
===========

To make the plugin work, the following steps are necessary:

#. Load the extension (Extension Manager).
#. Activate (inject) the extension (edit site template's Setup).
#. Configure the extension (edit site template's Constants).
#. Configure some pages for secure access (via Page or List module).
